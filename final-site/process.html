PLP Project: POINT

The point, line, plane project involved using nothing but elementary forms to create repeating patterns or designs that use tension. The sequence itself shows how an interesting image can be made from something as basic as points arranged at different sizes.

PLP Project: LINE

The point, line, plane project involved using nothing but elementary forms to create repeating patterns or designs that use tension. The sequence itself shows how an interesting image can be made from something as basic as one line formed from lines of varying sizes.

SYMBOL Project: POINT

The symbol project involved using simple forms to create a simple, abstract symbol. One design makes use of circles to create a square, patterned form, which retains its visibility when shrunk down.

SYMBOL Project: LINE

The symbol project involved using simple forms to create a simple, abstract symbol.

Connections Project

The connections project required creating a design that captured a certain data set. I initially tried to make a complex design fit with the information I had gathered. In the end I had to revisit and redo the design with simpler forms. The end result better achieved and visually represented the concept of momentum that defined my data set.

H Letterform Project

Compositions from the letterform H project feature pieces of the H form at different angles. In the various series, the legs of the H are at similar angles, but in various locations on the pages. This element brings interest to the final composition.

ANIMATION Project

The animation project allowed us to use the After Effects program to turn one of our previous point, line, plane series into a motion graphic. To convert a static project into a motion graphic, we had to consider new variables of time and sound. In my animation, I experimented with alternating two stills and using shapes for transitioning.

BOOKLET Project

In creating a booklet, I chose to use the typography project for the content. The most challenging aspect of this project was arranging the spreads in both a designer and print layout. The print layout had to be arranged in the correct signatures, front and back. Additionally, I used four signatures all bound together for my booklet.
